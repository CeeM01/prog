El juego de la vida.

profe aca le agrego mi implementacion del juego de la vida, utilice como lenguaje java e IDE processing ya que ayuda a la creacion de programas sin siquiera importar graficadores

mi juego tiene varias utilidades con diferentes teclas para poder manjearlo y entenderlo de una mejor manera:

TECLAS:

R = reinicia el juego y coloca celulas al azar
V = limpia el tablero
M = permite usarlo de manera manual y colocar celulas donde se le apetesca a quien maneje el codigo
